베이스라인 그리드

모든 구성요소는 8dp 사각형 베이스라인 그리드에 정렬됩니다. 글자는 4dp의 베이스라인 그리드에 정렬됩니다. 툴바의 아이콘은 4dp 사각형 베이스라인 그리드에 정렬됩니다. 이것은 모바일, 태블릿, 데스크탑에 적용됩니다.

키라인과 간격

모바일

모바일 레이아웃 템플릿은 어떻게 키라인과 간격이 스크린 가장자리와 요소에 걸쳐 작용하는지에 관한 아주 다양한 스크린과 정보를 포함하고 있습니다. 다음은 다운로드 파일에 포함된 스크린 샘플입니다.

Mobile Layout Template
2.4 MB (.ai)

태블릿

테블릿 레이아웃 템플릿은 14개의 서로 다른 레이아웃을 포함하고 있고, 키라인과 간격이 스크린 가장자리와 요소들 사이에서 어떻게 작용하는지 보여줍니다. 다음은 다운로드에 포함된 14개 스크린 중 두개의 샘플입니다.
Tablet Layout Template
3.19 MB (.ai)

데스크탑

데스크탑 레이아웃 템플릿은 각각 4개의 서로 다른 창 사이즈와 어떻게 키라인과 간격이 창과 요소들 사이에서 작용하는지를 보여주는 4개의 다른 화면을 포함합니다. 다음은 다운로드에서 사용 가능한 스크린 샘플 하나입니다. 윈도우 사이즈를 토대로, 키라인과 간격 블록은 태블릿과 모바일의 그리드 규칙을 따르게 됩니다.

비율 키라인

비율 키라인

모바일 스크린의 폭과 모바일, 태블릿, 데스크탑의 UI 요소의 폭에 적용하세요.

증가형 키라인

증가형 키라인

증가형 키라인은 액션바 높이같은 증가량을 규정하거나, 그러한 증가를 사용하여 앱 안의 다른 요소의 크기나 위치를 결정합니다.

대부분 데스크탑에 적용되고, 종종 태블릿이나 드물게 모바일에 적용되기도 합니다. 증가할 배수는 창 크기를 토대로 달라집니다.

터치 타겟 사이즈

최소한의 터치 타겟 사이즈는 48dp입니다. 아이콘(24dp)이나 아바타(40dp)의 간격을 레이아웃에서 조정할때 이것을 유념하세요. 터치 타겟은 겹쳐져서는 안됩니다.
